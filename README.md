## 💡 Sobre o Projeto

Um jogo de quiz interativo desenvolvido com **HTML, CSS e JavaScript puro** — sem frameworks, sem bibliotecas, só o básico bem feito.

O jogador responde 5 perguntas de múltipla escolha, recebe feedback visual imediato em cada resposta e ao final descobre sua pontuação com uma mensagem personalizada.

---

## ✨ Funcionalidades

- ✅ Perguntas dinâmicas geradas via JavaScript
- ✅ Feedback visual imediato (verde = certo, vermelho = errado)
- ✅ Barra de progresso animada
- ✅ Placar em tempo real
- ✅ Tela de resultado com mensagem personalizada
- ✅ Botão de reiniciar o quiz

```bash

# Entre na pasta
cd quiz-game

# Abra no navegador

open index.html   # Mac/Linux
```
